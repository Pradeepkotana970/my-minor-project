# AI-Driven Smart Monitoring System Using Face Recognition and Suspicious Activity Detection

## Abstract
The AI-Driven Smart Monitoring System is an advanced surveillance solution that combines real-time face recognition and intelligent behavior analysis to enhance security. The system identifies authorized individuals, detects unknown persons, analyzes suspicious activities, and generates automated alerts. It also provides a web-based dashboard for live monitoring, alert management, and analytics.

## Features
- Real-time face detection and recognition
- Unknown person detection
- Suspicious activity analysis
- Automated alert generation
- Live video streaming dashboard
- User registration and authentication
- Database logging and analytics

## Technologies Used
- Python
- OpenCV
- Flask
- SQLite
- HTML, CSS, JavaScript
- LBPH Face Recognizer
